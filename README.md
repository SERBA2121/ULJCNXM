- Wipe dalvik, cache, metadata,
- Flash ROM.zip,
- Wipe dalvik, cache, metadata,
- You can update the realking kernel or install NOKernel if you want,
- Format data,
- Reboot in fastboot,
- Execute fastboot erase frp,
- Execute fastboot -w,
- Reboot,
- Restart your device after completing the initial setup,
